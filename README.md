# 🎬 Movie Recommendation System

A C++ console-based Movie Recommendation System built using Data Structures and Algorithms (DSA).

## Features

- Add Movie
- Display Movies
- Search Movie (Linear Search)
- Recommend Movies by Genre
- Sort Movies by Rating (Bubble Sort)
- Delete Movie
- Show Top Rated Movie
- Save Movies to File
- Load Movies Automatically

## DSA Concepts Used

- Vector
- Structure
- Linear Search
- Bubble Sort
- File Handling

## Language

- C++

## How to Run

Compile:

```bash
g++ main.cpp movie.cpp -o movie
```

Run:

```bash
./movie
```
